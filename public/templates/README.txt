Cấu trúc thư mục (theo thứ tự sử dụng):

  01-sach-tai-lieu/     → Mẫu nhập Sách & Tài liệu (sách, giáo trình, luận văn...)
  02-tac-gia/           → Mẫu nhập Tác giả
  03-nha-xuat-ban/      → Mẫu nhập Nhà xuất bản
  04-the-loai/          → Mẫu nhập Thể loại sách (chủ đề)
  05-ngon-ngu/          → Mẫu nhập Ngôn ngữ tài liệu
  06-ban-doc/           → Mẫu nhập Bạn đọc (sinh viên, giảng viên...)

Trong mỗi thư mục có file CSV mẫu. Tải file tương ứng, điền dữ liệu theo đúng
cột, sau đó dùng chức năng "Import Excel" trên từng trang quản lý tương ứng.
